# SignalRChat_with-group

SignalR(ASP.NET)のマイクロソフト公式チュートリアルに作成した学習目的プロジェクトをグループレベルでの管理を可能にしたもの。

![サンプル画像](/dev/data/sample.gif)

## 開発環境

| 機能 | バージョン |
| ---- | ---- |
| Windows | 11 Home |
| Visual Studio | 2022 |
| .NET | 6.0 |
| C# | .NET依存 |
